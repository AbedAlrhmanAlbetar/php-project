# PHP Project

A full-stack web application that connects users with local service providers.

## Tech Stack
- Laravel (Backend)
- React + Vite (Frontend)

## Features
- Authentication system
- Service providers profiles
- Reviews and ratings
- Search and filtering
- 
## How to Run

### Backend (Laravel)
- cd backend
- composer install
- php artisan serve

### Frontend (React)
- cd frontend
- npm install
- npm run dev
